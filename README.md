# 🎨 Yurim Portfolio (React + GSAP)

React로 제작한 **개인 포트폴리오 웹사이트**입니다.  
**GSAP**으로 다양한 인터랙션/스크롤 애니메이션을 구현했고, 타이포그래피는 **나눔스퀘어(NanumSquare)** 폰트를 사용했습니다.  
**100% 개인 제작**, **반응형(Responsive)** 지원, **GitHub Pages**로 배포했습니다.

> 🔗 Live: https://yurim-web.github.io/yurim_portfolio_/

---

## ✨ 주요 특징

- **React 기반 SPA**
- **GSAP 애니메이션**(로드/스크롤/호버/타임라인)
- **나눔스퀘어 폰트**로 일관된 타이포그래피
- **반응형 웹** (모바일/태블릿/데스크톱)
- 포트폴리오 내부에 **직접 제작한 8개 사이트** 링크 연결
- 깔끔한 디렉터리 구조와 컴포넌트 분리

---

## 🧰 기술 스택

- **Languages:** HTML, CSS, JavaScript (ES6+)
- **Framework / Library:** React (JSX), GSAP
- **Scaffold:** Create React App
- **Package Manager:** npm
- **Deploy:** GitHub Pages

---

## 🚀 시작하기 (Local)

> Node.js 18+ 권장

```bash
# 1) 의존성 설치
npm install

# (GSAP 미설치 시)
npm install gsap

# 2) 개발 서버 실행 (http://localhost:3000)
npm start

# 3) 프로덕션 빌드
npm run build
```
