# Lee Yu Rim - Frontend Portfolio

React + GSAP 기반의 개인 포트폴리오 웹사이트입니다.

## Demo

https://yurim-web.github.io/yurimss_portfolio/

## Tech Stack

- React
- GSAP (ScrollTrigger)
- CSS (반응형 웹)

## Features

- 스크롤 기반 인터랙티브 애니메이션
- 반응형 웹 (PC / Mobile)
- 프로필, 강점, 경력, 포트폴리오, 연락처 섹션 구성

## Getting Started

```bash
npm install
npm start
```
